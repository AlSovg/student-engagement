# Risk Score Algorithm

Эвристический алгоритм оценки риска снижения вовлечённости студента. Реализован как чистая функция `calculateRisk()` в `src/lib/risk.ts`.

## Формула

Итоговый риск-скор — сумма весов сработавших факторов, ограниченная 100.

| Фактор         | Условие                                                  | Вес |
| -------------- | -------------------------------------------------------- | --- |
| Падение тренда | Индекс упал > 10 пунктов за последнюю неделю             | +40 |
| Низкий индекс  | Текущий индекс < 40                                      | +30 |
| Нет заданий    | 0 событий типа `ASSIGNMENT_SUBMIT` за последние 2 недели | +20 |
| Редкие входы   | Менее 4 входов (`LOGIN`) за последние 2 недели           | +10 |

**Максимум**: 100.

## Уровни риска

| Скор   | Уровень | Цвет    |
| ------ | ------- | ------- |
| 0–30   | Низкий  | Зелёный |
| 31–60  | Средний | Жёлтый  |
| 61–100 | Высокий | Красный |

## Входные данные

```ts
calculateRisk(
  scoreHistory: { score: number; period: Date }[], // история индексов (asc)
  activities:   { type: string; createdAt: Date }[], // активности студента
  now?: Date                                          // текущая дата (для тестов)
): RiskResult
```

`scoreHistory` берётся из таблицы `EngagementScore` за последние 4–8 недель.  
`activities` берётся из таблицы `Activity` за последние 4 недели.

## Где используется

| Страница         | Аудитория     | Отображение                                              |
| ---------------- | ------------- | -------------------------------------------------------- |
| `/dashboard`     | Преподаватель | Колонка «Риск» в таблице: бейдж + % для каждого студента |
| `/students/[id]` | Преподаватель | Карточка с уровнем, процентом и списком факторов         |
| `/me`            | Студент       | Карточка с уровнем, процентом и мотивирующим сообщением  |

## Мотивирующие сообщения (для студентов)

- **Низкий**: «Отличная работа! Вы активно участвуете в курсах и своевременно выполняете задания.»
- **Средний**: «Есть куда расти. Старайтесь чаще заходить на платформу и не откладывайте сдачу заданий.»
- **Высокий**: «Уровень вовлечённости вызывает беспокойство. Выполняйте задания, просматривайте материалы, участвуйте в обсуждениях.»
