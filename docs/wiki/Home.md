# Система мониторинга вовлеченности студентов

Веб-платформа для отслеживания цифровой активности студентов и визуализации их уровня вовлеченности в учебный процесс.

## Разделы

- [API Documentation](API-Documentation) — описание всех API эндпоинтов
- [Engagement Algorithm](Engagement-Algorithm) — алгоритм расчёта индекса вовлеченности
- [Developer Guide](Developer-Guide) — как запустить и разрабатывать проект

## Стек

| Слой               | Технология                  |
| ------------------ | --------------------------- |
| Frontend + Backend | Next.js 16 (App Router)     |
| База данных        | PostgreSQL + Prisma 7       |
| Аутентификация     | NextAuth.js v5              |
| UI                 | Tailwind CSS v4 + shadcn/ui |
| Язык               | TypeScript                  |
