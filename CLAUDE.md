# Система мониторинга вовлеченности студентов

## О проекте

Система анализирует цифровой след студентов (активность на платформе, время сессий, выполнение заданий, участие в обсуждениях) и визуализирует уровень вовлеченности для преподавателей.

Тип: студенческий учебный проект (магистратура, ИСиТ).
Цель: рабочий MVP + проектная документация.

## Стек

- **Frontend + Backend**: Next.js 15 (App Router, API Routes, Server Actions)
- **БД**: PostgreSQL + Prisma ORM
- **UI**: Tailwind CSS + shadcn/ui
- **Аутентификация**: NextAuth.js
- **Аналитика**: recharts или Chart.js для дашборда
- **Язык**: TypeScript

## Структура проекта

```
src/
  app/
    (auth)/         # страницы входа/регистрации
    dashboard/      # дашборд преподавателя
    students/       # профили студентов
    api/            # API routes
  components/
  lib/
    db.ts           # Prisma client
    auth.ts         # NextAuth config
  types/
prisma/
  schema.prisma
```

## Соглашения

- Компоненты — PascalCase, файлы — kebab-case
- API routes возвращают `{ data, error }`
- Серверные компоненты по умолчанию, `"use client"` только при необходимости
- Язык кода — английский, комментарии — русский (учебный проект)

## Ключевые сущности

- `User` (студент / преподаватель)
- `Course` (курс)
- `Activity` (событие цифрового следа: вход, просмотр материала, сдача задания)
- `EngagementScore` (расчётный показатель вовлеченности)

## Приоритеты MVP

1. Авторизация (студент / преподаватель)
2. Генерация/импорт тестовых данных активности
3. Расчёт индекса вовлеченности
4. Дашборд с графиками и списком студентов
5. Профиль студента с историей активности
